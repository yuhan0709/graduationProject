# graduationProject
毕业设计

## 项目启动
- git clone https://github.com/yuhan0709/graduationProject.git
- npm install
- cd web
    - npm install
    - npm run dev

- cd platform
    - npm install
    - npm run dev

## 介绍
本项目采用前后端分离思想，网络请求逻辑如下
1. `web`端发送请求至`node`端
2. `node`端发送请求至后台(如`severless`)
